## What I learned

I built this component to strengthen my skills in semantic HTML and CSS Grid.

With this project, I focused on creating a more elastic and fluid layout instead of relying only on rigid media queries. I practiced using `clamp()` for responsive typography and structured the layout using CSS Grid to ensure the design adapts naturally between mobile and desktop sizes.
